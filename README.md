## Умный реестр подписок (AI-агент)

Backend-ядро с AI-агентом на базе LangChain, который отвечает на вопросы о подписках:
- подсчёт трат за период в любой валюте,
- определение самой дорогой категории,
- поиск платежей на неделе.

Агент использует ReAct-подход, вызывая инструменты `get_obligations` и `convert_currency`, и логирует весь цикл рассуждений.

## Причины выбора:

1. Отличное качество на русском языке — DeepSeek показывает результаты, сравнимые с GPT-4.
2. Цена — в 10-15 раз дешевле GPT-4o при сопоставимом качестве рассуждений.
3. Function Calling / Tool Use — нативно поддерживает вызов инструментов, что критично для ReAct-агентов.
4. Длинный контекст (128K токенов) — позволяет обрабатывать большие списки обязательств без потери контекста.
6. Стабильность — низкий процент галлюцинаций благодаря строгому следованию инструкциям в промпте.
7. Возможность использования без VPN и сторонних сервисов.

## Мотивация участия в проекте

1. Что тебя привлекает в этом проекте?

Привлекает возможность получить опыт реальной промышленной разработки — не учебный проект, а работающий сервис с AI-агентом, который будет использоваться в компании. 
Хочется пройти полный цикл: от проектирования архитектуры до запуска в Docker, с тестами, логами и документацией.
Также ценно, что результат моей работы будет полезен бизнесу — агент реально поможет пользователям экономить время на анализе подписок и принимать финансовые решения.
В то же время, я смогу прокачать ключевые навыки: работу LLM, проектирование промптов, интеграцию с внешними API, контейнеризацию, написание тестов.
Наконец, хочется поработать в команде: научиться согласовывать архитектурные решения, проводить код-ревью, видеть, как мой вклад вписывается в общий результат, и расти вместе с коллегами.

2. Как ты видишь свою роль в команде?

AI/ML разработчик. Готов взять на себя:

1. Проектирование и реализацию инструментов агента (получение данных, конвертация валют).
2. Написание и отладку промптов для ReAct-цикла.
3. Интеграцию с внешними API
4. Документирование архитектуры и логики работы агента.
